Entity Relationship Creator

This web application will let you easily create computer drawn Entity Relationship boxes to get your ER-Diagram started. It can difficult to visualize both directions of the relationship along with the relationship description and the cardinality symbols.

Here you can build each direction of the relation in easy to understand text. Once everything is filled out the way you like you can submit the form and have the application return an SVG graphic that you can use as-is or import into a graphic application for further construction.
